# www.SebCode.com

## Technologies utilisées
REACT, Bootstrap, jQuery, JavaScript, JSON, CSS, HTML5, .NET 5.0, C#, WebApi, Swagger, Entity Framework Core 5.0, Azure Database (MS SQL), Azure App Services, Azure AD B2C, Azure DevOps (CI/CD), Azure Service Domains, Git, SendGrid.

## Internationalisation

### i18next-parser

*i18next-parser* est utilisé pour balayer les composantes REACT et générer les traductions.
https://github.com/i18next/i18next-parser

Pour générer ou rafraîchir les traductions, exécuter:
*"i18n parse.cmd"*.

### i18n-manager

*i18n-manager* est utilisé pour gérer les fichiers de traduction.
https://github.com/gilmarsquinelato/i18n-manager
